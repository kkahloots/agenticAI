# Knowledge Agent Specification\n\n## Overview\n\nThe Knowledge Agent is responsible for information retrieval and synthesis. It handles customer profile lookups, product catalog queries, and knowledge base searches using RAG (Retrieval-Augmented Generation).\n\n## Implementation\n\n- **File**: `agentic/knowledge_agent.py`\n- **Function**: `knowledge_agent(state: AgenticState) -> dict`\n- **Autonomy Level**: Low (follows clear retrieval patterns)\n\n## Core Responsibilities\n\n### 1. Customer Data Retrieval\n- **Customer Profiles**: Full customer information via MCP\n- **Identity Status**: KYC and identity verification status\n- **Customer Context**: Historical data and preferences\n\n### 2. Product Catalog Access\n- **Product Information**: Product details and specifications\n- **Category Browsing**: Product categories and hierarchies\n- **Search Functionality**: Product search and filtering\n\n### 3. Knowledge Base Search\n- **Semantic Search**: ChromaDB-based vector search\n- **Multi-Collection**: Search across emails, notes, policies\n- **RAG Generation**: LLM-synthesized answers with citations\n\n### 4. Answer Synthesis\n- **Context Integration**: Combine multiple data sources\n- **Citation Management**: Proper source attribution\n- **Response Formatting**: Human-readable output\n\n## MCP Server Integration\n\n### Customer Data Server\n- **Server ID**: `customer_data`\n- **Tools Used**:\n  - `search_customer_profile`: Get full customer profile\n  - `get_identity_status`: Identity verification status\n  - `get_kyc_status`: KYC compliance status\n\n### Product Catalog Server\n- **Server ID**: `product_catalog`\n- **Tools Used**:\n  - `get_products`: Retrieve product information\n  - `search_products`: Product search functionality\n  - `get_categories`: Product category information\n\n### Knowledge Base (Direct Integration)\n- **Function**: `multi_query_search`\n- **Collections**: emails, notes, policies\n- **Technology**: ChromaDB vector search\n\n## Decision Logic\n\n### Request Routing Decision Tree\n\n```python\ndef determine_retrieval_strategy(request: str) -> str:\n    customer_id = _extract_customer_id(request)\n    request_lower = request.lower()\n    \n    # Customer-specific queries\n    if customer_id and any(w in request_lower for w in [\"profile\", \"customer\", \"identity\", \"kyc\"]):\n        return \"customer_data\"\n    \n    # Product queries\n    elif any(w in request_lower for w in [\"product\", \"catalog\", \"item\"]):\n        return \"product_catalog\"\n    \n    # Knowledge base search\n    else:\n        return \"knowledge_base\"\n```\n\n### Customer ID Extraction\n```python\ndef _extract_customer_id(text: str) -> str | None:\n    \"\"\"Extract customer ID from text using regex.\"\"\"\n    match = re.search(r\"CUST-\\d+\", text, re.IGNORECASE)\n    return match.group(0).upper() if match else None\n```\n\n## Retrieval Strategies\n\n### 1. Customer Data Retrieval\n\n#### Profile Lookup\n```python\n# MCP call to customer data server\nresult = customer_server.invoke_tool(\n    \"search_customer_profile\", \n    customer_id=customer_id\n)\n\n# Expected result structure\n{\n    \"customer\": {\n        \"customer_id\": \"CUST-001\",\n        \"segment\": \"vip\",\n        \"identity_status\": \"verified\",\n        \"fraud_score\": 0.126,\n        \"engagement_score\": 0.6,\n        \"lifetime_value\": 15000.0,\n        \"country\": \"USA\",\n        \"last_interaction_date\": \"2025-01-15\"\n    }\n}\n```\n\n#### Identity Status Check\n```python\n# MCP call for identity verification\nresult = customer_server.invoke_tool(\n    \"get_identity_status\",\n    customer_id=customer_id\n)\n\n# Expected result structure\n{\n    \"customer_id\": \"CUST-001\",\n    \"identity_status\": \"verified\",\n    \"verification_date\": \"2024-03-15\",\n    \"expiry_date\": \"2026-03-15\",\n    \"documents_required\": []\n}\n```\n\n### 2. Product Catalog Retrieval\n\n#### Product Listing\n```python\n# MCP call to product catalog server\nresult = product_server.invoke_tool(\n    \"get_products\",\n    limit=10,\n    category=category_filter\n)\n\n# Expected result structure\n{\n    \"products\": [\n        {\n            \"product_id\": \"PROD-001\",\n            \"name\": \"Premium Widget\",\n            \"category\": \"electronics\",\n            \"price\": 299.99,\n            \"availability\": \"in_stock\"\n        }\n    ],\n    \"total_count\": 150\n}\n```\n\n### 3. Knowledge Base Search\n\n#### Multi-Collection Search\n```python\n# Direct function call to knowledge base\nresult = multi_query_search(query=request)\n\n# Expected result structure\n{\n    \"chunks\": [\n        {\n            \"text\": \"Policy content...\",\n            \"source\": \"fraud-policy.md\",\n            \"doc_type\": \"policy\",\n            \"relevance_score\": 0.85,\n            \"collection\": \"policies\"\n        }\n    ],\n    \"query\": \"original query\",\n    \"total_results\": 5\n}\n```\n\n## Answer Synthesis\n\n### Customer Profile Synthesis\n```python\ndef _synthesize_customer_answer(result: dict) -> str:\n    if \"customer\" in result:\n        c = result[\"customer\"]\n        if c:\n            return (\n                f\"Customer: {c['customer_id']} | Segment: {c.get('segment')} | \"\n                f\"Identity: {c.get('identity_status')} | Fraud score: {c.get('fraud_score')} | \"\n                f\"Engagement: {c.get('engagement_score')}\"\n            )\n        return \"Customer not found\"\n```\n\n### Product Catalog Synthesis\n```python\ndef _synthesize_product_answer(result: dict) -> str:\n    if \"products\" in result:\n        products = result[\"products\"]\n        if products:\n            return f\"Found {len(products)} products: \" + \", \".join(\n                p.get(\"product_id\", \"unknown\") for p in products[:5]\n            )\n        return \"No products found\"\n```\n\n### RAG-Based Synthesis\n```python\ndef _rag_generate(question: str, chunks: list[dict]) -> str:\n    \"\"\"Synthesize grounded answer from retrieved chunks using LLM.\"\"\"\n    context = \"\\n\\n\".join(\n        f\"[{c.get('doc_type', 'doc')}] {c.get('source', 'unknown')}:\\n{c.get('text', '')[:400]}\"\n        for c in chunks\n    )\n    \n    system_prompt = (\n        \"You are a helpful assistant. Answer the question using the provided sources. \"\n        \"Cite the source filename in your answer. \"\n        \"If the sources contain relevant information, summarize it clearly.\"\n    )\n    \n    llm = get_llm(temperature=0.0)\n    response = llm.invoke([\n        SystemMessage(content=system_prompt),\n        HumanMessage(content=f\"Sources:\\n{context}\\n\\nQuestion: {question}\")\n    ])\n    \n    return response.content.strip()\n```\n\n## State Updates\n\n### Input State Requirements\n- `original_request`: User's request text\n- `user_id`: User identifier for memory\n- `session_id`: Session identifier for conversation\n- `request_id`: Unique request ID for audit\n\n### Output State Updates\n```python\n{\n    \"active_agent\": \"knowledge\",\n    \"agent_history\": [\"knowledge\"],\n    \"mcp_calls\": [\n        {\n            \"server\": \"customer_data\",\n            \"tool\": \"search_customer_profile\",\n            \"params\": {\"customer_id\": \"CUST-001\"},\n            \"result\": result_data\n        }\n    ],\n    \"intermediate_results\": [\n        {\"agent\": \"knowledge\", \"result\": result_data}\n    ],\n    \"final_result\": result_data,\n    \"messages\": [AIMessage(content=synthesized_answer)],\n    \"audit_trail\": [\n        {\n            \"agent\": \"knowledge\",\n            \"action\": \"retrieve_information\",\n            \"mcp_calls\": 1\n        }\n    ]\n}\n```\n\n## Memory Integration\n\n### Memory Updates\n```python\n# Record conversation\nmemory_manager.conversation.add_message(\n    state[\"session_id\"], \n    \"assistant\", \n    synthesized_answer\n)\n\n# Record interaction\nmemory_manager.interaction.record_interaction(\n    state[\"user_id\"],\n    \"knowledge_retrieval\",\n    {\n        \"request\": request,\n        \"answer\": synthesized_answer,\n        \"sources\": source_list\n    }\n)\n```\n\n## Error Handling\n\n### MCP Server Errors\n```python\ntry:\n    result = customer_server.invoke_tool(\"search_customer_profile\", customer_id=customer_id)\nexcept Exception as e:\n    return {\n        \"error\": f\"Customer data server error: {str(e)}\",\n        \"fallback_action\": \"knowledge_base_search\"\n    }\n```\n\n### Knowledge Base Errors\n```python\ntry:\n    result = multi_query_search(query=request)\nexcept Exception as e:\n    return {\n        \"error\": f\"Knowledge base search error: {str(e)}\",\n        \"fallback_response\": \"I'm unable to search the knowledge base at the moment.\"\n    }\n```\n\n### Synthesis Errors\n```python\ntry:\n    answer = _rag_generate(request, chunks)\nexcept Exception as e:\n    # Fallback to simple chunk listing\n    lines = [\n        f\"[{i+1}] [{c.get('source', 'unknown')}] {c.get('text', '')[:200]}\"\n        for i, c in enumerate(chunks)\n    ]\n    answer = \"**Sources:**\\n\" + \"\\n\".join(lines)\n```\n\n## Performance Considerations\n\n### Optimization Strategies\n1. **Customer ID Caching**: Cache frequently accessed customer profiles\n2. **Product Catalog Caching**: Cache product information with TTL\n3. **Vector Search Optimization**: Optimize ChromaDB queries\n4. **LLM Response Caching**: Cache RAG responses for common queries\n\n### Latency Targets\n- **Customer Profile Lookup**: < 200ms\n- **Product Catalog Query**: < 300ms\n- **Knowledge Base Search**: < 500ms\n- **RAG Generation**: < 1000ms\n- **Total Knowledge Agent Time**: < 1500ms\n\n## Testing\n\n### Unit Tests\n- Customer ID extraction accuracy\n- MCP server integration\n- Answer synthesis quality\n-
