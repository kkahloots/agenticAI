# Workflow Agent Specification\n\n## Overview\n\nThe Workflow Agent handles complex multi-step processes that require coordination between multiple agents and systems. It decomposes high-level goals into executable sub-tasks and orchestrates their execution.\n\n## Implementation\n\n- **File**: `agentic/other_agents.py`\n- **Function**: `workflow_agent(state: AgenticState) -> dict`\n- **Autonomy Level**: High (makes complex orchestration decisions)\n\n## Core Responsibilities\n\n### 1. Goal Decomposition\n- **Task Analysis**: Break down complex goals into sub-tasks\n- **Dependency Mapping**: Identify task dependencies and ordering\n- **Resource Planning**: Determine required agents and tools\n- **Timeline Estimation**: Estimate execution time and milestones\n\n### 2. Agent Coordination\n- **Sub-Agent Invocation**: Call specialized agents for sub-tasks\n- **State Management**: Maintain workflow state across agent calls\n- **Result Aggregation**: Combine results from multiple agents\n- **Error Propagation**: Handle failures in sub-tasks\n\n### 3. Progress Monitoring\n- **Milestone Tracking**: Monitor progress against planned milestones\n- **Status Reporting**: Provide workflow status updates\n- **Bottleneck Detection**: Identify and resolve workflow bottlenecks\n- **Performance Metrics**: Track workflow execution metrics\n\n### 4. Adaptive Execution\n- **Dynamic Replanning**: Adjust workflow based on intermediate results\n- **Resource Reallocation**: Reassign resources based on needs\n- **Failure Recovery**: Implement recovery strategies for failed sub-tasks\n- **Optimization**: Optimize workflow execution in real-time\n\n## MCP Server Integration\n\n### Primary MCP Servers\n- **CRM Server**: For customer relationship management operations\n- **Analytics Server**: For data analysis and insights\n- **Customer Data Server**: For customer information access\n\n### Workflow-Specific Tools\n- **Workflow Management**: Track and manage workflow state\n- **Task Scheduling**: Schedule and coordinate sub-tasks\n- **Resource Monitoring**: Monitor resource usage and availability\n\n## Decision Logic\n\n### Workflow Type Classification\n```python\ndef classify_workflow_type(request: str) -> str:\n    \"\"\"Classify the type of workflow based on the request.\"\"\"\n    request_lower = request.lower()\n    \n    # Campaign management workflows\n    if any(keyword in request_lower for keyword in [\"campaign\", \"marketing\", \"promotion\"]):\n        return \"campaign_management\"\n    \n    # Customer engagement workflows\n    elif any(keyword in request_lower for keyword in [\"engagement\", \"retention\", \"churn\"]):\n        return \"customer_engagement\"\n    \n    # Business process workflows\n    elif any(keyword in request_lower for keyword in [\"process\", \"workflow\", \"automation\"]):\n        return \"business_process\"\n    \n    # Analysis and reporting workflows\n    elif any(keyword in request_lower for keyword in [\"report\", \"analysis\", \"insights\"]):\n        return \"analysis_reporting\"\n    \n    # Default to general workflow\n    else:\n        return \"general_workflow\"\n```\n\n### Goal Decomposition Strategy\n```python\ndef decompose_goal(goal: str, workflow_type: str) -> dict:\n    \"\"\"Decompose high-level goal into executable sub-tasks.\"\"\"\n    \n    if workflow_type == \"customer_engagement\":\n        return {\n            \"sub_tasks\": [\n                {\n                    \"task_id\": \"analyze_customer_segments\",\n                    \"agent\": \"analytics\",\n                    \"description\": \"Analyze customer segments and identify at-risk customers\",\n                    \"dependencies\": [],\n                    \"estimated_time\": 300  # Sequential Execution\n```python\ndef execute_sequential_workflow(workflow_plan: dict, state: AgenticState) -> dict:\n    \"\"\"Execute workflow tasks sequentially.\"\"\"\n    results = []\n    current_state = state.copy()\n    \n    for task in workflow_plan[\"sub_tasks\"]:\n        # Check dependencies\n        if not all_dependencies_completed(task[\"dependencies\"], results):\n            continue\n        \n        # Execute task\n        task_result = execute_sub_task(task, current_state)\n        results.append(task_result)\n        \n        # Update state with task results\n        current_state = merge_task_result(current_state, task_result)\n        \n        # Check for early termination conditions\n        if should_terminate_early(task_result, workflow_plan):\n            break\n    \n    return aggregate_workflow_results(results, workflow_plan)\n```\n\n### 2. Parallel Execution\n```python\ndef execute_parallel_workflow(workflow_plan: dict, state: AgenticState) -> dict:\n    \"\"\"Execute independent workflow tasks in parallel.\"\"\"\n    # Identify parallelizable tasks\n    parallel_groups = identify_parallel_groups(workflow_plan[\"sub_tasks\"])\n    \n    all_results = []\n    \n    for group in parallel_groups:\n        if len(group) == 1:\n            # Single task - execute normally\n            result = execute_sub_task(group[0], state)\n            all_results.append(result)\n        else:\n            # Multiple tasks - execute in parallel\n            parallel_results = execute_tasks_in_parallel(group, state)\n            all_results.extend(parallel_results)\n    \n    return aggregate_workflow_results(all_results, workflow_plan)\n```\n\n### 3. Adaptive Execution\n```python\ndef execute_adaptive_workflow(workflow_plan: dict, state: AgenticState) -> dict:\n    \"\"\"Execute workflow with dynamic adaptation based on intermediate results.\"\"\"\n    results = []\n    current_plan = workflow_plan.copy()\n    \n    while has_remaining_tasks(current_plan):\n        # Select next task based on current state\n        next_task = select_next_task(current_plan, results, state)\n        \n        # Execute task\n        task_result = execute_sub_task(next_task, state)\n        results.append(task_result)\n        \n        # Evaluate intermediate results\n        evaluation = evaluate_intermediate_results(results, current_plan)\n        \n        # Adapt plan if necessary\n        if evaluation[\"should_replan\"]:\n            current_plan = replan_workflow(current_plan, results, evaluation)\n        \n        # Update state\n        state = merge_task_result(state, task_result)\n    \n    return aggregate_workflow_results(results, current_plan)\n```\n\n## Sub-Task Execution\n\n### Agent Invocation\n```python\ndef execute_sub_task(task: dict, state: AgenticState) -> dict:\n    \"\"\"Execute a single sub-task by invoking the appropriate agent.\"\"\"\n    agent_name = task[\"agent\"]\n    task_id = task[\"task_id\"]\n    \n    # Prepare task-specific state\n    task_state = prepare_task_state(state, task)\n    \n    try:\n        # Invoke the appropriate agent\n        if agent_name == \"analytics\":\n            from agentic.analytics_agent import analytics_agent\n            result = analytics_agent(task_state)\n        elif agent_name == \"recommendation\":\n            from agentic.recommendation_agent import recommendation_agent\n            result = recommendation_agent(task_state)\n        elif agent_name == \"action\":\n            result = action_agent(task_state)\n        elif agent_name == \"evaluation\":\n            result = evaluation_agent(task_state)\n        else:\n            raise ValueError(f\"Unknown agent: {agent_name}\")\n        \n        # Wrap result with task metadata\n        return {\n            \"task_id\": task_id,\n            \"agent\": agent_name,\n            \"status\": \"completed\",\n            \"result\": result,\n            \"execution_time\": calculate_execution_time(),\n            \"timestamp\": datetime.now().isoformat()\n        }\n        \n    except Exception as e:\n        return {\n            \"task_id\": task_id,\n            \"agent\": agent_name,\n            \"status\": \"failed\",\n            \"error\": str(e),\n            \"execution_time\": calculate_execution_time(),\n            \"timestamp\": datetime.now().isoformat()\n        }\n```\n\n##
