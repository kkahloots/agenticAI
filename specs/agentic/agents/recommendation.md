# Recommendation Agent Specification\n\n## Overview\n\nThe Recommendation Agent generates personalized recommendations using machine learning models. It handles collaborative filtering, content-based recommendations, hybrid approaches, and cold start scenarios.\n\n## Implementation\n\n- **File**: `agentic/recommendation_agent.py`\n- **Function**: `recommendation_agent(state: AgenticState) -> dict`\n- **Autonomy Level**: Medium (makes ML model selection decisions)\n\n## Core Responsibilities\n\n### 1. Recommendation Generation\n- **Collaborative Filtering**: User-user and item-item similarity\n- **Content-Based**: Product feature similarity\n- **Hybrid Models**: Weighted combination of approaches\n- **Cold Start Handling**: Recommendations for new users/items\n\n### 2. Model Selection\n- **Strategy Selection**: Choose optimal recommendation strategy\n- **Model Switching**: Adapt based on data availability\n- **Performance Optimization**: Select best-performing models\n- **Fallback Mechanisms**: Handle model failures gracefully\n\n### 3. Personalization\n- **User Profile Integration**: Leverage user preferences and history\n- **Context Awareness**: Consider current session and behavior\n- **Dynamic Adaptation**: Adjust recommendations based on feedback\n- **Preference Learning**: Learn from user interactions\n\n### 4. Ranking and Filtering\n- **Relevance Scoring**: Score recommendations by relevance\n- **Diversity Optimization**: Ensure recommendation diversity\n- **Business Rules**: Apply business constraints and rules\n- **Explanation Generation**: Provide recommendation explanations\n\n## MCP Server Integration\n\n### Recommendation Server\n- **Server ID**: `recommendation`\n- **Tools Used**:\n  - `recommend_products`: Generate ML-based recommendations\n  - `get_similar_customers`: Find similar users for collaborative filtering\n  - `get_content_recommendations`: Content-based recommendations\n  - `get_hybrid_recommendations`: Hybrid model recommendations\n  - `explain_recommendations`: Generate recommendation explanations\n\n### Customer Data Server\n- **Server ID**: `customer_data`\n- **Tools Used**:\n  - `search_customer_profile`: Get user profile for personalization\n  - `get_interaction_history`: Retrieve user interaction history\n\n### Product Catalog Server\n- **Server ID**: `product_catalog`\n- **Tools Used**:\n  - `get_products`: Retrieve product information\n  - `get_product_features`: Get product attributes for content-based\n\n## Decision Logic\n\n### Recommendation Strategy Selection\n\n```python\ndef select_recommendation_strategy(customer_id: str, context: dict) -> str:\n    \"\"\"Select optimal recommendation strategy based on user data.\"\"\"\n    \n    # Get user interaction history\n    interaction_count = get_user_interaction_count(customer_id)\n    \n    # Cold start scenario\n    if interaction_count < 5:\n        return \"popularity_based\"\n    \n    # Limited data - use content-based\n    elif interaction_count < 20:\n        return \"content_based\"\n    \n    # Sufficient data for collaborative filtering\n    elif interaction_count < 100:\n        return \"collaborative_filtering\"\n    \n    # Rich data - use hybrid approach\n    else:\n        return \"hybrid\"\n```\n\n### Model Performance Evaluation\n```python\ndef evaluate_model_performance(customer_id: str, recommendations: list) -> dict:\n    \"\"\"Evaluate recommendation quality metrics.\"\"\"\n    return {\n        \"diversity_score\": calculate_diversity(recommendations),\n        \"novelty_score\": calculate_novelty(recommendations, customer_id),\n        \"coverage_score\": calculate_catalog_coverage(recommendations),\n        \"confidence_score\": calculate_confidence(recommendations)\n    }\n```\n\n## Recommendation Strategies\n\n### 1. Collaborative Filtering\n\n#### User-Based Collaborative Filtering\n```python\n# MCP call to recommendation server\nresult = recommendation_server.invoke_tool(\n    \"get_similar_customers\",\n    customer_id=customer_id,\n    similarity_threshold=0.7,\n    max_neighbors=50\n)\n\n# Expected result structure\n{\n    \"customer_id\": \"CUST-001\",\n    \"similar_customers\": [\n        {\n            \"customer_id\": \"CUST-002\",\n            \"similarity_score\": 0.85,\n            \"common_products\": [\"PROD-001\", \"PROD-003\"]\n        }\n    ],\n    \"recommendations\": [\n        {\n            \"product_id\": \"PROD-005\",\n            \"score\": 0.92,\n            \"reason\": \"Customers similar to you also liked this\"\n        }\n    ]\n}\n```\n\n#### Item-Based Collaborative Filtering\n```python\n# Get user's recent interactions\nuser_products = get_user_recent_products(customer_id, limit=10)\n\n# Find similar products\nresult = recommendation_server.invoke_tool(\n    \"get_similar_products\",\n    product_ids=user_products,\n    similarity_method=\"cosine\",\n    max_recommendations=20\n)\n```\n\n### 2. Content-Based Recommendations\n\n#### Feature-Based Similarity\n```python\n# MCP call for content-based recommendations\nresult = recommendation_server.invoke_tool(\n    \"get_content_recommendations\",\n    customer_id=customer_id,\n    feature_weights={\n        \"category\": 0.4,\n        \"brand\": 0.3,\n        \"price_range\": 0.2,\n        \"rating\": 0.1\n    },\n    max_recommendations=15\n)\n\n# Expected result structure\n{\n    \"customer_id\": \"CUST-001\",\n    \"user_preferences\": {\n        \"preferred_categories\": [\"electronics\", \"books\"],\n        \"preferred_brands\": [\"Apple\", \"Samsung\"],\n        \"price_range\": {\"min\": 50, \"max\": 500}\n    },\n    \"recommendations\": [\n        {\n            \"product_id\": \"PROD-010\",\n            \"score\": 0.88,\n            \"matching_features\": [\"category\", \"brand\"],\n            \"reason\": \"Matches your preference for electronics from Apple\"\n        }\n    ]\n}\n```\n\n### 3. \"get_cold_start_recommendations\",\n    customer_id=customer_id,\n    strategy=\"popularity_with_demographics\",\n    demographic_filters={\n        \"country\": user_country,\n        \"age_group\": user_age_group\n    },\n    max_recommendations=12\n)\n\n# Expected result structure\n{\n    \"customer_id\": \"CUST-NEW\",\n    \"cold_start_strategy\": \"popularity_with_demographics\",\n    \"demographic_match\": {\n        \"country\": \"USA\",\n        \"age_group\": \"25-34\"\n    },\n    \"recommendations\": [\n        {\n            \"product_id\": \"PROD-020\",\n            \"popularity_score\": 0.89,\n            \"demographic_relevance\": 0.76,\n            \"reason\": \"Popular among users in your demographic\"\n        }\n    ]\n}\n```\n\n## Recommendation Processing\n\n### Result Ranking and Filtering\n```python\ndef rank_and_filter_recommendations(recommendations: list, user_profile: dict) -> list:\n    \"\"\"Apply business rules and ranking to recommendations.\"\"\"\n    \n    # Filter out already purchased items\n    filtered_recs = [r for r in recommendations if not user_has_purchased(r[\"product_id\"])]\n    \n    # Apply business rules\n    business_filtered = apply_business_rules(filtered_recs, user_profile)\n    \n    # Ensure diversity\n    diverse_recs = ensure_diversity(business_filtered, diversity_threshold=0.3)\n    \n    # Final ranking\n    ranked_recs = sorted(diverse_recs, key=lambda x: x[\"final_score\"], reverse=True)\n    \n    return ranked_recs[:10]  # Top 10 recommendations\n```\n\n### Explanation Generation\n```python\ndef generate_explanations(recommendations: list, strategy: str) -> list:\n    \"\"\"Generate human-readable explanations for recommendations.\"\"\"\n    explained_recs = []\n    \n    for rec in recommendations:\n        explanation = {\n            \"product_id\": rec[\"product_id\"],\n            \"score\": rec[\"final_score\"],\n            \"primary_reason\": get_primary_reason(rec, strategy),\n            \"detailed_explanation\": generate_detailed_explanation(rec, strategy),\n            \"confidence\": rec.get(\"confidence_score\", 0.8)\n        }\n        explained_recs.append(explanation)\n    \n    return explained_recs\n```\n\n### Recommendation Formatting\n```python\ndef format_recommendation_results(result: dict, strategy: str) -> str:\n    \"\"\"Format recommendation results for human consumption.\"\"\"\n    recommendations = result.get(\"recommendations\", [])\n    customer_id = result.get(\"customer_id\")\n    \n    if not recommendations:\n        return f\"No recommendations found for customer {customer_id}.\"\n    \n    summary = f\"Top {len(recommendations)} recommendations for {customer_id} (using {strategy}):\\n\\n\"\n    \n    for i, rec in enumerate(recommendations[:5], 1):\n        product_id = rec.get(\"product_id\")\n        score = rec.get(\"final_score\", rec.get(\"score\", 0))\n        reason = rec.get(\"reason\", rec.get(\"explanation\", \"Recommended for you\"))\n        \n        summary += f\"{i}. {product_id} (Score: {score:.2f})\\n\"\n        summary += f\"   Reason: {reason}\\n\\n\"\n    \n    if len(recommendations) > 5:\n        summary += f\"... and {len(recommendations) - 5} more recommendations\\n\"\n    \n    return summary\n```\n\n## State Updates\n\n###
