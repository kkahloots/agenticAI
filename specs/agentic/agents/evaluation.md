# Evaluation Agent Specification\n\n## Overview\n\nThe Evaluation Agent assesses workflow outcomes, provides feedback, and triggers replanning when results don't meet expectations. It serves as the quality control and continuous improvement mechanism for the agentic system.\n\n## Implementation\n\n- **File**: `agentic/other_agents.py`\n- **Function**: `evaluation_agent(state: AgenticState) -> dict`\n- **Autonomy Level**: High (makes critical evaluation and replanning decisions)\n\n## Core Responsibilities\n\n### 1. Outcome Evaluation\n- **Success Measurement**: Evaluate if goals were achieved\n- **Quality Assessment**: Assess the quality of results\n- **Performance Analysis**: Analyze execution performance metrics\n- **Deviation Detection**: Identify significant deviations from expectations\n\n### 2. Feedback Generation\n- **Structured Feedback**: Provide detailed evaluation feedback\n- **Improvement Suggestions**: Suggest specific improvements\n- **Root Cause Analysis**: Identify causes of suboptimal outcomes\n- **Learning Capture**: Extract learnings for future improvements\n\n### 3. Replanning Decisions\n- **Replan Triggering**: Decide when replanning is necessary\n- **Strategy Adjustment**: Recommend strategy changes\n- **Resource Reallocation**: Suggest resource adjustments\n- **Timeline Modification**: Adjust execution timelines\n\n### 4. Continuous Learning\n- **Pattern Recognition**: Identify patterns in successes and failures\n- **Best Practice Extraction**: Extract successful patterns as best practices\n- **Failure Analysis**: Analyze failure modes and prevention strategies\n- **Knowledge Base Updates**: Update system knowledge with learnings\n\n## MCP Server Integration\n\n### Analytics Server\n- **Server ID**: `analytics`\n- **Tools Used**:\n  - `calculate_success_metrics`: Calculate outcome success metrics\n  - `analyze_performance_trends`: Analyze performance over time\n  - `detect_anomalies`: Identify unusual patterns in results\n  - `compare_benchmarks`: Compare results against benchmarks\n\n### Evaluation-Specific Tools\n- **Outcome Measurement**: Measure specific business outcomes\n- **Quality Scoring**: Score result quality using various metrics\n- **Deviation Analysis**: Analyze deviations from expected outcomes\n\n## Decision Logic\n\n### Evaluation Criteria Selection\n```python\ndef select_evaluation_criteria(workflow_type: str, original_goal: str) -> dict:\n    \"\"\"Select appropriate evaluation criteria based on workflow type.\"\"\"\n    \n    base_criteria = {\n        \"execution_success\": {\"weight\": 0.3, \"threshold\": 0.8},\n        \"time_efficiency\": {\"weight\": 0.2, \"threshold\": 0.7},\n        \"resource_utilization\": {\"weight\": 0.2, \"threshold\": 0.6}\n    }\n    \n    # Add workflow-specific criteria\n    if workflow_type == \"customer_engagement\":\n        base_criteria.update({\n            \"engagement_improvement\": {\"weight\": 0.4, \"threshold\": 0.1},  # 10% improvement\n            \"customer_satisfaction\": {\"weight\": 0.3, \"threshold\": 0.8}\n        })\n    elif workflow_type == \"campaign_management\":\n        base_criteria.update({\n            \"conversion_rate\": {\"weight\": 0.4, \"threshold\": 0.05},  # 5% conversion\n            \"roi\": {\"weight\": 0.3, \"threshold\": 2.0}  # 2x ROI\n        })\n    elif workflow_type == \"analytics_reporting\":\n        base_criteria.update({\n            \"data_accuracy\": {\"weight\": 0.4, \"threshold\": 0.95},\n            \"insight_quality\": {\"weight\": 0.3, \"threshold\": 0.8}\n        })\n    \n    return base_criteria\n```\n\n### Success Assessment\n```python\ndef assess_workflow_success(results: dict, criteria: dict) -> dict:\n    \"\"\"Assess overall workflow success based on criteria.\"\"\"\n    \n    criterion_scores = {}\n    weighted_score = 0.0\n    total_weight = 0.0\n    \n    for criterion_name, criterion_config in criteria.items():\n        weight = criterion_config[\"weight\"]\n        threshold = criterion_config[\"threshold\"]\n        \n        # Calculate criterion score\n        actual_value = results.get(criterion_name, 0)\n        if criterion_name in [\"engagement_improvement\", \"conversion_rate\", \"roi\"]:\n            # Higher is better\n            score = min(1.0, actual_value / threshold) if threshold > 0 else 0\n        else:\n            # Achievement-based scoring\n            score = 1.0 if actual_value >= threshold else actual_value / threshold\n        \n        criterion_scores[criterion_name] = {\n            \"score\": score,\n            \"actual_value\": actual_value,\n            \"threshold\": threshold,\n            \"met_threshold\": score >= 1.0\n        }\n        \n        weighted_score += score * weight\n        total_weight += weight\n    \n    overall_score = weighted_score / total_weight if total_weight > 0 else 0\n    \n    return {\n        \"overall_score\": overall_score,\n        \"criterion_scores\": criterion_scores,\n        \"success_level\": determine_success_level(overall_score),\n        \"met_all_thresholds\": all(c[\"met_threshold\"] for c in criterion_scores.values())\n    }\n```\n\n### Replanning Decision\n```python\ndef determine_replanning_need(assessment: dict, workflow_context: dict) -> dict:\n    \"\"\"Determine if replanning is needed based on assessment.\"\"\"\n    \n    overall_score = assessment[\"overall_score\"]\n    replan_count = workflow_context.get(\"replan_count\", 0)\n    max_replans = workflow_context.get(\"max_replans\", 2)\n    \n    # Don't replan if already replanned too many times\n    if replan_count >= max_replans:\n        return {\n            \"should_replan\": False,\n            \"reason\": f\"Maximum replans ({max_replans}) reached\",\n            \"recommendation\": \"Accept current results or escalate to human\"\n        }\n    \n    # Replan if overall score is too low\n    if overall_score < 0.6:\n        return {\n            \"should_replan\": True,\n            \"reason\": f\"Overall score ({overall_score:.2f}) below threshold (0.6)\",\n            \"replan_strategy\": determine_replan_strategy(assessment)\n        }\n    \n    # Replan if critical criteria failed\n    critical_failures = identify_critical_failures(assessment)\n    if critical_failures:\n        return {\n            \"should_replan\": True,\n            \"reason\": f\"Critical criteria failed: {', '.join(critical_failures)}\",\n            \"replan_strategy\": \"address_critical_failures\"\n        }\n    \n    # No replanning needed\n    return {\n        \"should_replan\": False,\n        \"reason\": \"Results meet acceptable thresholds\",\n        \"recommendation\": \"Proceed with current results\"\n    }\n```\n\n## Evaluation Strategies\n\n### 1. Quantitative Evaluation\n\n#### Metric-Based Assessment\n```python\n# MCP call to calculate success metrics\nresult = analytics_server.invoke_tool(\n    \"calculate_success_metrics\",\n    workflow_id=workflow_id,\n    metrics=[\n        \"completion_rate\",\n        \"execution_time\",\n        \"resource_efficiency\",\n        \"outcome_quality\"\n    ],\n    time_period=\"workflow_duration\"\n)\n\n# Expected result structure\n{\n    \"workflow_id\": \"WF-001\",\n    \"metrics\": {\n        \"completion_rate\": {\n            \"value\": 0.85,\n            \"benchmark\": 0.80,\n            \"performance\": \"above_benchmark\"\n        },\n        \"execution_time\": {\n            \"value\": 1200,  # Qualitative Evaluation\n\n#### Quality Assessment\n```python\ndef assess_result_quality(results: dict, quality_dimensions: list) -> dict:\n    \"\"\"Assess qualitative aspects of results.\"\"\"\n    \n    quality_scores = {}\n    \n    for dimension in quality_dimensions:\n        if dimension == \"completeness\":\n            # Assess if all expected outputs were produced\n            expected_outputs = results.get(\"expected_outputs\", [])\n            actual_outputs = results.get(\"actual_outputs\", [])\n            score = len(actual_outputs) / len(expected_outputs) if expected_outputs else 1.0\n            \n        elif dimension == \"accuracy\":\n            # Assess accuracy of results\n            accuracy_metrics = results.get(\"accuracy_metrics\", {})\n            score = accuracy_metrics.get(\"overall_accuracy\", 0.8)\n            \n        elif dimension == \"relevance\":\n            # Assess relevance to original goal\n            relevance_score = calculate_relevance_score(results, original_goal)\n            score = relevance_score\n            \n        elif dimension == \"timeliness\":\n            # Assess if results were delivered on time\n            actual_time = results.get(\"execution_time\", 0)\n            expected_time = results.get(\"expected_time\", actual_time)\n            score = min(1.0, expected_time / actual_time) if actual_time > 0 else 1.0\n        \n        quality_scores[dimension] = {\n            \"score\": score,\n            \"grade\": assign_quality_grade(score)\n        }\n    \n    overall_quality = sum(qs[\"score\"] for qs in quality_scores.values()) / len(quality_scores)\n    \n    return {\n        \"quality_dimensions\": quality_scores,\n        \"overall_quality\": overall_quality,\n        \"quality_grade\": assign_quality_grade(overall_quality)\n    }\n```\n\n### 3. data_points=workflow_metrics,\n    detection_method=\"statistical\",\n    sensitivity=\"medium\",\n    historical_baseline=\"last_30_workflows\"\n)\n\n# Expected result structure\n{\n \"high\",\n            \"estimated_impact\": \"Significant improvement potential\"\n        })\n    \n    return recommendations\n```\n\n## State Updates\n\n### Input State Requirements\n-
