Context retrieval operations\n    if any(keyword in request_lower for keyword in [\"context\", \"history\", \"previous\", \"earlier\"]):\n        return \"context_retrieval\"\n    \n    # Profile management operations\n    elif any(keyword in request_lower for keyword in [\"preference\", \"profile\", \"setting\", \"remember\"]):\n        return \"profile_management\"\n    \n    # Interaction recording operations\n    elif any(keyword in request_lower for keyword in [\"record\", \"save\", \"store\", \"log\"]):\n        return \"interaction_recording\"\n    \n    # Knowledge and learning operations\n    elif any(keyword in request_lower for keyword in [\"learn\", \"observe\", \"insight\", \"pattern\"]):\n        return \"knowledge_management\"\n    \n    # Recency scoring (more recent = more relevant)\n    time_diff = calculate_time_difference(context_item[\"timestamp\"])\n    recency_score = max(0, 1.0 - (time_diff / 3600))  # Decay over 1 hour\n    relevance_score += recency_score * 0.3\n    \n    # Semantic similarity scoring\n    semantic_score = calculate_semantic_similarity(\n        context_item[\"content\"], \n        current_request\n    )\n    relevance_score += semantic_score * 0.4\n    \n    # Topic continuity scoring\n    topic_score = calculate_topic_continuity(\n        context_item.get(\"topics\", []),\n        extract_topics(current_request)\n    )\n    relevance_score += topic_score * 0.3\n    \n    return min(1.0, relevance_score)\n```\n\n### Memory Retention Policy\n```python\ndef apply_retention_policy(memory_type: str, data_age_days: int) -> dict:\n    \"\"\"Apply data retention policies based on memory type and age.\"\"\"\n    \n    retention_policies = {\n        \"conversation\": {\n            \"max_age_days\": 30,\n            \"archive_threshold_days\": 7,\n            \"auto_summarize\": True\n        },\n        \"user_profile\": {\n            \"max_age_days\": 365,\n            \"archive_threshold_days\": 90,\n            \"auto_summarize\": False\n        },\n        \"interaction\": {\n            \"max_age_days\": 180,\n            \"archive_threshold_days\": 30,\n            \"auto_summarize\": True\n        },\n        \"agent_observation\": {\n            \"max_age_days\": 90,\n            \"archive_threshold_days\": 30,\n            \"auto_summarize\": True\n        }\n    }\n    \n    policy = retention_policies.get(memory_type, retention_policies[\"conversation\"])\n    \n    if data_age_days > policy[\"max_age_days\"]:\n        return {\"action\": \"delete\", \"reason\": \"Exceeded maximum retention period\"}\n    elif data_age_days > policy[\"archive_threshold_days\"]:\n        return {\"action\": \"archive\", \"reason\": \"Exceeded active retention period\"}\n    else:\n        return {\"action\": \"keep\", \"reason\": \"Within active retention period\"}\n```\n\n## Memory Operations\n\n### 1.
