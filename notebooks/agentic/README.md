## What's Different?

The agentic system features:
- **Dynamic Routing**: Context-aware agent selection
- **MCP Servers**: Standardized tool protocol
- **Memory Layer**: 4 types of memory (conversation, profile, interaction, observation)
- **Evaluation**: Continuous outcome assessment with replanning
- **Collaboration**: Agents work together and share state

## Comparison Notebook

The `comparison.ipynb` notebook shows:
- Same requests executed by both systems
- Output comparison
- Performance metrics
- Architecture differences
- Key benefits of agentic approach
